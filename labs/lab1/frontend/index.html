What is a smart contract?

Read this first

A smart contract is a program stored on the blockchain. It is not an agreement between two people. It is public code that any participant on the network can call.

SimpleStorage (the contract running right now on your local chain) stores exactly two things:

  • favoriteNumber — a number, visible to everyone on the chain
  • lessonMessage — a text string, also public and readable by anyone

When you submit the form below, you send a transaction to the network. The network executes it and permanently updates those two values. The left side of the page immediately reflects the new state. Anyone connected to this chain sees the same change.

The contract is not between two people. The network enforces it. The state belongs to everyone on the chain simultaneously. Think of it as a shared spreadsheet that no single person can edit secretly — every write is a public, permanent transaction.
